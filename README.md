# lrn

A Markdown Note taking TUI made in Rust obv

<img src="./image.png" />

How to install

1. Clone the repository to your local machine using

```bash
git clone git@github.com:lhagfoss/lrn.git
```

2. Ensure you have Cargo installed on your system.

```bash
cargo --version
```

3. Navigate to the project directory

```bash
cd lrn
```

4. Install the application using Cargo with the command

```bash
cargo install --path .
```

5. Run the application by typing `lrn` in your terminal.

```bash
lrn
```
